# Backend-101

Practical implementations of common backend concepts using **FastAPI**.

## Projects

1. **[Rate Limiter](Rate-Limiter/README.md)**

   Rate limiting strategies implemented using FastAPI and Redis, including fixed window, sliding window, and token bucket approaches.

2. **[Redis Caching](Redis-Caching/README.md)**

   FastAPI Product Catalog API using Redis for implementation of the cache-aside pattern, cache invalidation, and anti-stampede locking.
